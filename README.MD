## 설치

- node version: v9.0.0^
- npm version: v6.0.0^

```
$ npm install
```

## dev에서 실행하기

```
$ npm run dev
```

- http://localhost:8080 접속

## Proudction 빌드 및 배포

```
$ npm run build
$ npm start
```

- http://localhost:3000.
